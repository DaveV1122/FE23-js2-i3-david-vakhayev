# FE23-js2-i3-david-vakhayev

Dummy webshop built with TypeScript, Parcel and DummyJSON.

## Features

- Fetches products from DummyJSON
- Shows product cards
- Search products
- Shows image, title, description, rating, stock, category
- Shows low-stock warning when stock is below 10
- Includes an Add to cart button

## Run locally

```bash
npm install
npm run dev
```

## Build for GitHub Pages

```bash
npm run build
```
